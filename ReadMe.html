Øxide Installation Instructions

Follow these steps carefully to install Øxide Panel into Gorilla Tag:

  1. Make sure you have a Gorilla Tag mod loader installed (like BepInEx).
  2. Locate your Gorilla Tag installation folder. Usually it's in C:\Program Files (x86)\Steam\steamapps\common\Gorilla Tag.
  3. Inside the Gorilla Tag folder, find the Plugins directory. If it doesn’t exist, create a folder named Plugins.
  4. Download the Øxide panel.dll file from the Download page.
  5. Move or copy the Øxide panel.dll file into the Plugins folder.
  6. Launch Gorilla Tag. Øxide Panel should automatically load with your mod menu.
  7. In-game, open the Øxide menu using your configured keybind and start exploring all the features and fun mods.

Tips: Always make a backup of your Plugins folder before adding new mods. If the panel doesn’t show, make sure your mod loader is up-to-date.
